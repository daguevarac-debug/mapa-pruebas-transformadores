# mapa-pruebas-transformadores

Proyecto personal y educativo para estudiar y organizar procedimientos de ensayo de transformadores de potencia como mapa de conocimiento.

## Principios de esta etapa

- El objetivo es educativo. Esta herramienta no se usara para operar laboratorio, aprobar resultados ni liberar transformadores.
- Los procedimientos internos en PDF ubicados en la raiz "Procedimientos Campo de pruebas" son fuentes externas de solo lectura.
- Toda informacion tecnica debe conservar trazabilidad documental.
- Una afirmacion no puede marcarse como `verified` sin referencia documental concreta (archivo y pagina o seccion).
- Todavia no se ha seleccionado tecnologia para la aplicacion.
